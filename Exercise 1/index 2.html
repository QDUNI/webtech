What will you learn?

Iterating with loops

An essential part of programming are loops. With loops you can iterate through a list or a sequence. You learn what type of loops exist and how to use them.

Data types

Programming always involves data. The kind of data you use is specific to the application of your program. You learn what type of data exist and when to use them.

Complex Algorithms

To solve a problem or to perform a calculation, you use algorithms. In this course you learn to analyse and make complex algorithms.
